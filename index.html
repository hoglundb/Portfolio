Brian Hoglund's Portfolio
Here is a compilation of some of the computer graphics projects and games that I have completed. These projects feature the use of a wide range of graphics technologies including Open GL, Unity, C#, C++ and HTML/Javascript.
Disclaimer: This is a collection of my personal game development-related work. My professional AR/VR projects contain information controlled by the DoD classification guidelines and are unavailable for public viewing.
Flight Simulation Multiplayer Game (Unity)
A multiplayer flight simulation game that I made for my senior project at Oregon State University. This game was made with the Unity Game Engine and I collaborated with Wes Bakane and Joe Kirkhan. We build this game over a 12 Week period and implemented realistic physics/ballistics as well as a local multiplayer feature.
You can download the game here: https://hoglundb.itch.io/flight-sim-multiplayer-game
Ocean Shader Program (Unity URP)
A small shader program that I wrote using Unity's Univeral Render Pipeline. It uses a simple vertex shader to simulate the motion of the waves and a fragment shader to animate the surface of the water. This is a prototype for a game I am currently working on, with the goal of eventually simulating boyancy physics.
You can view a Web GL build of the simulation here: https://simmer.io/@Brian_Hoglund/oceansimtest
Zombie Survival Game (Unity)
This is a game I developed in colaboration with Wes Bakane for the Ludum Dare 48 2021 Game Jam. We built this game over a 2 day period using the Unity Game Engine. The goal of the game is to use your archery skills to survive the waves of zombies and to make your way to the bottom of the pit, where you will find the zombie cure.
You can download the game here: https://bakanew.itch.io/the-depth-of-death
Simple Zombie Game (Open GL)
This was a simple c++ zombie game that I wrote using Open GL and GLSL. The goal of the game is to survice waves of zombies by punching them.
You can view the source code for this project here https://github.com/hoglundb/OpenGLZombieGame
Network Visualization (Javascript/HTML5)
A web application that I developed as part of a research project with TeachEngineering.org. It allows users to view the Next Generation Science Standards (NGSS) in a network format. This application allows K-12 curriculumn providers to visually analyze how their content aligns with the NGSS.
You can view the website here: https://www.teachengineering.org/ngss_visualization
